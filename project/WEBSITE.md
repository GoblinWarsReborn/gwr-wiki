# Website - Especificação de Páginas

## Menu Principal

O site conta com a seguinte estrutura de navegação:

1. Home
2. Whitepaper
3. Wiki
4. Marketing Place
5. My Goblins
6. Tokens
7. Login via Metamask

---

## 1. Home

A página inicial apresenta as seguintes seções:

### Seções da Página
- **Hero Banner**: Destaque do jogo mobile com opções de download
  - Botão para download Android
  - Botão para download iOS
- **Minning Mode**: Informações sobre o modo de mineração
- **Arena Mode**: Informações sobre o modo arena
- **Stock Options**: Opções de investimento
- **Roadmap**: Planejamento e evolução do projeto
- **Redes Sociais**: Links para as redes sociais do projeto

---

## 2. Whitepaper

Página dedicada à documentação técnica do projeto.

- **Conteúdo**: Apenas texto
- **Objetivo**: Apresentar detalhes técnicos, econômicos e conceituais do projeto

---

## 3. Wiki

Página de documentação e guias do jogo.

- **Conteúdo**: Apenas texto
- **Objetivo**: Fornecer guias, tutoriais e informações sobre mecânicas do jogo

---

## 4. Marketing Place

Marketplace para negociação de itens e goblins.

- **Modelo**: Similar ao CryptoKitties
- **Funcionalidades**:
  - Compra e venda de goblins
  - Listagem de itens disponíveis
  - Sistema de leilões/ofertas
  - Histórico de transações

---

## 5. My Goblins

Área pessoal para gerenciamento dos goblins do jogador.

### Minha Lista de Goblins
- Visualização de todos os goblins que o jogador possui
- Filtros e opções de ordenação

### Página do Goblin
Ao selecionar um goblin específico, exibe:
- **Atributos**: Força, Agilidade, Vigor, Inteligência, Carisma
- **Genética**: Informações sobre a genética do goblin
- **Observação**: Os itens equipados não são exibidos nesta página (apenas no app mobile)

---

## 6. Tokens

Página para gerenciamento financeiro e tokens do jogo.

### Funcionalidades
- **Compra de Tokens**: Interface para aquisição de tokens do jogo
- **Acompanhamento de Saldo**: Visualização do saldo atual
- **Saque**: Opção para retirada de tokens

---

## 7. Login

Sistema de autenticação descentralizada.

- **Método**: Login via Metamask
- **Integração**: Conexão com carteira Web3 para autenticação segura
